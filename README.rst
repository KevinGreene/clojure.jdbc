# clj.jdbc

Alternative implementation of jdbc wrapper for clojure.


## Install

Leiningen

```clojure
[be.niwi/clj.jdbc "0.1.0-beta1"]
```

Graddle

```groovy
compile "be.niwi:clj.jdbc:0.1.0-beta1"
```

Maven

```xml
<dependency>
  <groupId>be.niwi</groupId>
  <artifactId>clj.jdbc</artifactId>
  <version>0.1.0-beta1</version>
</dependency>
```
